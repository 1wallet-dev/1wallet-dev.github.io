One Wallet

Simple

to create a wallet, just scan a QR code using the Google Authenticator app

Secure

authorize transactions with 6-digit code from Google Authenticator. No private keys or passwords to take care of.

Durable

easily restore wallet by scanning QR code exported by Google Authenticator, or recover funds using another wallet.

Smart

configurable spending limit, composable security, and auto-recover mechanisms. Imagine a (physical) wallet that has an embedded computer - it can do a lot more than a plain old wallet that only carries your money around.
